# Plataforma Web para ONGs de Proteção Animal

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como trabalho acadêmico com o objetivo de criar uma **plataforma web para ONGs de proteção animal**, facilitando a conexão entre organizações, voluntários e pessoas interessadas em adoção.

A proposta busca utilizar a tecnologia como ferramenta para **incentivar a adoção responsável, divulgar resgates e conscientizar a população sobre o cuidado com animais**.

---

## 🎯 Objetivos
- Criar um website funcional para ONGs
- Divulgar animais disponíveis para adoção
- Informar sobre resgates e cuidados com os animais
- Promover conscientização e engajamento social
- Centralizar informações importantes em uma única plataforma

---

## 🚀 Funcionalidades

🔹 **Página inicial (Home)**  
Apresentação da ONG e destaque para animais disponíveis

🔹 **Adoção de Animais**  
Listagem de pets com páginas individuais contendo detalhes

🔹 **Resgates**  
Informações sobre animais resgatados e suas histórias

🔹 **Blog**  
Conteúdos informativos sobre cuidados com animais

🔹 **FAQ**  
Perguntas frequentes para auxiliar usuários

🔹 **Vacinação**  
Informações sobre cuidados e saúde dos animais

🔹 **Portfólio**  
Apresentação de envolvidos no projeto/ONG
